# 需求分析输出模板

用于输出标准需求分析或交接给 `prd-generator`。根据实际信息裁剪，不要填充空洞内容。

## 澄清模式模板

```markdown
## 已理解的信息

- [已知事实]

## 关键缺口

| 缺口 | 影响 |
| --- | --- |
| [缺失信息] | [会影响范围/流程/依赖/数据/原型/验收] |

## 澄清问题

1. [问题]  
   影响：[说明为什么重要]

## 建议下一步

- [补充信息后进入需求分析/直接进入 PRD 生成]
```
